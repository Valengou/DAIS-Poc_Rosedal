# DAIS-POC Rosedal - Operations Dashboard

Este repositorio contiene los reportes y análisis de las operaciones de vuelo realizadas con drones.

## Contenido

- **Dashboard Global**: Vista consolidada de todas las operaciones
- **Mapas H3 Hexbin**: Visualización espacial de detecciones por clase
- **Análisis Temporal**: Mapa interactivo con slider temporal
- **Reporte Estadístico**: Análisis estadístico completo con gráficos

## Acceso

Los reportes están disponibles a través de GitHub Pages. Una vez configurado, accede a:

`https://[tu-usuario].github.io/[nombre-repo]/dashboard.html`

## Estructura

- `dashboard.html`: Dashboard principal con enlaces a todos los reportes
- `temporal_analysis_map.html`: Mapa temporal con análisis por hora
- `statistical_report.html`: Reporte estadístico completo
- `OP#*/h3_hexbin_map.html`: Mapas individuales por operación

## Tecnologías

- Leaflet.js para visualización de mapas
- H3 para indexación geoespacial
- Matplotlib para gráficos estadísticos
- Python para procesamiento de datos
